# [Octane][]

[![Version badge][]][version]
[![Build badge][]][build]

Octane parses [Rocket League][] replays.

## Usage

1.  Install [Stack][].

2.  `stack --resolver nightly install octane`

3.  `stack exec octane path/to.replay`
