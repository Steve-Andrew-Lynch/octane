# [Octane][]

[![Version badge][]][package version]
[![Build badge][]][build status]

Octane parses [Rocket League][] replays.

## Usage

1.  Install [Stack][].

2.  Install Octane with `stack build`.

3.  Parse a replay with `stack exec octane path/to/the.replay`.
